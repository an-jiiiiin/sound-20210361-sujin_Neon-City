# Neon Pulse City (p5.js Music Visualizer)

## Controls
- Click: Play / Pause
- Move mouse left-right: Pan (audio) + City shift (visual)
- N: Random neon palette
- 1 / 2: Switch tracks (Somebodyhelpus.mp3 / OneRe.mp3)
- H: Toggle UI

## Files
- index.html
- style.css
- sketch.js
- p5.js
- p5.sound.min.js
- Somebodyhelpus.mp3
- OneRe.mp3

## How to Run
좌측 상단 별 모양을 클릭해 1번 혹은 2번 음악을 재생하실 수 있습니다.
